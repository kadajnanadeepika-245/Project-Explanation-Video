## 🚀 Project 1: PRONUNEX – Intelligent Speech Therapy Platform

### Problem Statement
People with pronunciation difficulties often lack access to accurate and personalized speech evaluation systems.

### Objective
To develop an AI-powered speech therapy platform that evaluates pronunciation, identifies errors, and provides personalized practice recommendations.

### Technologies Used
- React.js
- Django REST Framework
- Python
- Librosa
- Wav2Vec2
- Whisper
- JWT Authentication

## 🌍 Project 2: AI-Powered Jharkhand Tourism

### Problem Statement
Many tourist destinations in Jharkhand are not widely known due to limited digital promotion and travel assistance.

### Objective
To build an AI-powered tourism platform that helps users explore tourist destinations with AI recommendations, maps, AR, and 360° virtual experiences.

### Technologies Used
- React.js
- TypeScript
- Vite
- Tailwind CSS
- Three.js
- React Three Fiber
- WebXR API
